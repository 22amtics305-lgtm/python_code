##### 6\)What is Loop in Python?



A loop is used to repeat a block of code multiple times.

###### Easy definition:-

A loop executes the same code again and again until a condition becomes false.

###### Types of Loops in Python:-

###### For Loop → Used when the number of repetitions is fixed.

###### syntax:-

for i in range(1, 6):

&#x20;   print(i)

###### While Loop → Used when repetition depends on a condition.

syntax:-

i = 1

while i <= 5:

&#x20;   print(i)

&#x20;   i += 1
